# Project-001: FieldRise AI秘書

## 目的

社長専属のAI秘書を構築し、「社長の時間を増やし、創造と経営に集中できる環境を作ること」を実現する。

## 管理対象

| 領域 | 内容 | 取得方法（想定） | 状態 |
|---|---|---|---|
| SoundOn | 音楽配信の実績・収益 | 要調査（API有無） | 未着手 |
| YouTube | チャンネル分析 | YouTube Data API | 未着手 |
| TikTok | 投稿・分析 | TikTok API | 未着手 |
| Instagram | 投稿・分析 | Instagram Graph API | 未着手 |
| 農業データ | 生育・作業記録 | 手動入力 + スプレッドシート | 未着手 |
| 天気情報 | 圃場周辺の天気予報 | 気象API（Open-Meteo等、キー不要） | 未着手 |
| 売上情報 | 音楽・米の売上 | 手動入力 + 集計自動化 | 未着手 |
| AIニュース | AI業界の最新動向 | RSS / ニュースAPI | 未着手 |
| 規約変更 | 各プラットフォームの規約監視 | 定期チェック | 未着手 |
| 改善提案 | COOからの自発的提案 | 定期レビュー | 未着手 |

## 方針（案）

第一段階として、認証不要で取得できる情報（天気、AIニュース）からGitHub Actionsによる自動収集を開始し、日次ブリーフィングを生成する。第二段階でYouTube等のAPI連携（GitHub Secretsでキー管理）を追加し、最終的に全10領域を統合したレポートを社長へ自動配信する。

本方針はCTO（ChatGPT）のレビュー待ち（`cto/inbox/2026-07-18_project-001-architecture-review.md` 参照）。
